# 🔍 Elite SEO Fundamentals (Search Engine Optimization)

Kode yang cantik tidak ada gunanya jika web lu terdampar di halaman 15 Google. *Engineer* sejati harus tahu cara membuat karya mereka disukai robot Google (Crawlers).

## 🎯 Tujuan Skill
Memastikan situs memiliki Semantik HTML (Struktur) yang benar dan kecepatan muat (*Load Time*) di bawah 2 detik untuk memenangkan persaingan halaman pertama Google.

## ⚠️ Kapan Digunakan
- Saat membuat blog, artikel publikasi, dan *Landing Page* perusahaan.

## ⛔ Larangan Keras (NEVER DO THIS)
1. **JANGAN PERNAH** menggunakan *tag* `<h1>` lebih dari satu kali dalam satu halaman.
2. **JANGAN PERNAH** membuat *website* berbasis JavaScript *Single Page Application* (React murni / Vue murni) jika kontennya adalah Blog Publik tanpa *Server-Side Rendering* (Next.js/Nuxt). Googlebot akan melihat halaman putih kosong!

## ✅ Aturan Wajib (ALWAYS DO THIS)
1. **Meta Title & Description**: WAJIB unik untuk setiap halaman. 
2. **Alt Image**: Robot Google itu buta. Deskripsikan gambar menggunakan tag `alt="..."` pada setiap `<img>`.
