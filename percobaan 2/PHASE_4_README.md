# Phase 4 — Agent Execution Package

## Purpose

This phase contains the final files to give directly to an AI coding agent.

It includes:

1. `PROTOTYPE_BUILD_PROMPT.md`  
   The direct build prompt.

2. `ACCEPTANCE_CRITERIA.md`  
   Checklist for whether the prototype is done.

3. `QA_TEST_PLAN.md`  
   Testing procedure for browser, mobile, gameplay, save/load, and performance.

4. `FINAL_AGENT_HANDOFF.md`  
   Final consolidated instruction file.

## Important

Before coding, the agent must read Phase 1, Phase 2, Phase 3, and Phase 4.

The most important rule:

> Build the first playable web prototype only. Do not build the full game.
